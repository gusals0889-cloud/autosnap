## 1. 프로젝트 소개

**AutoSnap** 은 Windows 환경에서 화면의 특정 영역을 빠르게 캡처하고, 화살표와 텍스트를 이용해 간단한 주석을 추가할 수 있는 데스크탑 유틸리티입니다.  
트레이 아이콘을 통해 언제든 캡처를 시작할 수 있으며, 저장된 이미지는 자동으로 `captures` 폴더에 정리됩니다.

---

## 2. 기획 및 개발 방향

AutoSnap는 다음과 같은 목표로 설계되었습니다.

- 빠른 화면 캡처
- 직관적인 주석 추가
- 최소한의 의존성으로 가벼운 실행
- exe 단독 실행 가능한 배포 형태 지원

사용자는 별도의 복잡한 편집 프로그램 없이도 캡처 후 바로 메모를 남기고 저장할 수 있습니다.

---

## 3. 주요 기능

| 기능 | 설명 |
|------|------|
| **영역 선택 캡처** | 투명 오버레이 창에서 마우스로 드래그하여 원하는 영역을 지정하고 캡처합니다. |
| **주석 도구** | 화살표와 텍스트를 추가해 이미지에 핵심 내용을 표시할 수 있습니다. |
| **시스템 트레이 연동** | 트레이 아이콘 메뉴를 통해 캡처 기능을 빠르게 실행할 수 있습니다. |
| **자동 저장** | 캡처 및 주석이 반영된 이미지를 PNG 형식으로 자동 저장합니다. |
| **설정 확장성** | 색상, 두께, 폰트 등 기본 요소를 코드 기반으로 쉽게 조정할 수 있습니다. |

## 6. 특장점

- **경량성**: `tkinter`와 `Pillow` 중심으로 구성해 가볍게 동작합니다.
- **즉시 사용 가능**: exe 파일만으로 실행할 수 있어 설치 부담이 적습니다.
- **직관적인 흐름**: 캡처 → 주석 → 저장의 과정을 단순하게 구성했습니다.
- **확장 가능성**: 도형, 색상 선택, 단축키 같은 기능을 쉽게 추가할 수 있습니다.

## 7. 한계점

- **Windows 전용**: 현재 구조상 Windows 환경에 맞춰져 있습니다.
- **기본 주석 기능 중심**: 화살표와 텍스트 위주의 단순한 편집에 초점이 맞춰져 있습니다.
- **고급 편집 기능 부족**: 레이어, 필터, 자유 변형 같은 기능은 제공하지 않습니다.
- **실행 중 리소스 사용**: 트레이와 캡처 로직이 백그라운드에서 동작합니다.

## 9. 사용 방법

### exe 파일로 실행하는 경우
1. Releases 페이지에서 `autosnap.exe` 파일을 다운로드합니다.
2. 실행하면 시스템 트레이에 아이콘이 표시됩니다.
3. 트레이 메뉴에서 **캡처 시작**을 선택합니다.
4. 원하는 영역을 선택하고 주석을 추가한 뒤 저장합니다.
5. 결과 이미지는 `captures` 폴더에 저장됩니다.

### Python으로 직접 실행하는 경우
```bash
pip install pillow pystray
python autosnap.py
```

1. 필요한 패키지를 설치합니다.
2. `autosnap.py`를 실행합니다.
3. 캡처와 주석 기능을 사용합니다.
